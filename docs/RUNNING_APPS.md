# Running an App

1. **Open an App**: Click the card image or 'Open App' button.
2. **Configure Parameters - Basic**: Adjust prompts, image size, seed, etc.
3. **Configure Parameters - Advanced**: Tune CFG, sampler, scheduler, etc. (if available).
4. **Configure Parameters - Masking**: Draw masks for inpainting Apps.
5. **Generate**: Click "Generate" to start. Click "Interrupt" to stop.
6. **Live Preview**: View progress in the preview window.
7. **Recent**: Access previously generated images in the Recent panel.
