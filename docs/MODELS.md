# Installing and Uninstalling Models

## Model Status
When selecting a category, cards indicate if required models are installed.

## Information Panel
Click the (i) button on an App card to access the information panel and view required models.

## Installing Models
If a model is missing (red 'Missing' badge), click the **Install** button below it to download and install.

## Uninstalling Models
If a model is not used by any App, you can delete it by clicking **Uninstall**.
