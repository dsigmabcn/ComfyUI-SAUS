# File Manager

A simple manager for files in your ComfyUI installation.

- **Favorited Folders**: Shortcuts to 'Input', 'Output', and 'Models'.
- **Upload File**: Upload files/models to local machine.
- **Download from URL**: Download via link (supports API tokens for CivitAI/Hugging Face).
- **Folder Navigation**: Navigate folders (limited to ComfyUI directory).
- **Image Visualization**: Preview image/video files.
- **File Management**: Rename, download, or delete files.
